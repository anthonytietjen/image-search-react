# Image Search

**Technologies**: React, React Context API, Pixabay API

**Demo Site**: https://anthonytietjen-imagesearch.herokuapp.com

## Running locally

1. Clone this repo
1. Create a .env file in the project root with the value of `PIXABAY_API_KEY=%your_key_here%`
1. npm install
1. npm run dev

## Running tests

- npm test
