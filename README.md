# Image Search

**Technologies**: React, React Context API, Pixabay API

**Demo Site**: https://anthonytietjen-imagesearch.vercel.app

## Running locally

1. Clone this repo
1. Create a .env fil in the project root with the value of `REACT_APP_PIXABAY_API_KEY=%your_key_here%`
1. npm install
1. npm start

## Running tests

- npm test
